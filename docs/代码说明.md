# 源码文件说明

| 文件 | 作用 |
|------|------|
| `grammar.h` | 全局定义：Token 类型码、四元式操作码、符号表/Token/四元式结构体、Compiler 全局上下文 |
| `scanner.c/h` | 词法分析器：关键字/界符查表、常量自动机识别数字、输出 Token 序列 |
| `symbol.c/h` | 符号表系统：标识符查填、常量查填、临时变量 t1~tN 分配、a1~a6 翻译文法语义栈 |
| `parser.c/h` | 语法+语义分析：递归下降语句分析、7 级优先级表达式分析、IF/WHILE 跳转四元式生成、翻译文法 a1~a6 |
| `quadruple.c/h` | 中间代码（四元式）：`(op,arg1,arg2,result)` 生成/输出/回填 |
| `optimize.c/h` | 优化器：常量折叠、常量传播、死代码消除，多遍迭代至不动点 |
| `codegen.c/h` | 目标代码生成：四元式 → x86-64 AT&T 汇编，RIP 寻址，printf 调用 |
| `main.c` | 入口：CLI 模式 或 GTK3 图形界面（五标签页），含读取文件/GCC 编译运行按钮 |

## 运行

```bash
# 编译
gcc $(pkg-config --cflags gtk+-3.0) src/*.c -o compiler $(pkg-config --libs gtk+-3.0) -lm

# 使用
./compiler                        # GTK3 GUI
./compiler test/sample1.txt       # CLI
```
