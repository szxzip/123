# 编译原理课程设计跟踪表

**题目:** 一个简单文法的编译器前端 + x86-64 目标代码生成

## 模块完成情况

| 模块 | 子项 | 完成 | 说明 |
|---|---|---|---|
| **文法** | If/While/复合语句 | ✓ | if-then-else, while-do, begin-end 嵌套 |
| | 数组/结构体/函数 | ✗ | |
| **词法** | 常数处理机 | ✓ | 8 状态 Pascal 自动机, 整数/实数/科学计数法 |
| | 字符常量/字符串常量 | ✗ | |
| **语法** | 递归下降子程序 | ✓ | PROGRAM→SUB_PROGRAM→VARIABLE→COM_SENTENCE→... |
| | 表达式分析 | ✓ | 7 级优先级递归下降 (or→and→not→比较→±→*/→因子) |
| | LL(1)/LR | ✗ | |
| **语义** | 四元式设计 | ✓ | 20 种操作码 (program/:=/±*/div/jmp/jnz/6种比较/and/or/not/label/write/end) |
| | 翻译文法 | ✓ | a1~a6 (变量声明: push→写符号表 FIFO), 表达式语义动作 |
| | 符号表设计 | ✓ | 名字/类型/种类/偏移/宽度 5 字段 + 常数表 + 临时变量 |
| | 活动记录 | ✓ | offset 从 0 递增, integer=4 real=8 char=1 |
| **优化** | 常值表达式节省 | ✓ | 常量折叠: `2*5`→`10`, 传播后再次折叠 |
| | 常量传播 | ✓ | `a:=2; b:=a+3`→`b:=5` |
| | 删除无用赋值 | ✓ | 临时变量使用计数=0 则移除 |
| | 基本块/循环优化 | ✗ | |
| **目标代码** | 目标指令集定义 | ✓ | x86-64 AT&T: mov/add/sub/imul/idiv/cmp+setX/jmp/jne/call |
| | 目标指令生成 | ✓ | 四元式→汇编逐条翻译, label 显式定义 |
| **其它** | 可视化界面 | ✓ | GTK3: 源代码输入 + 编译按钮 + Token/符号表/四元式 三标签页 |
| | 运行平台 | ✓ | CLI + Linux x86-64, `gcc -no-pie x.s -o x && ./x` |

## 源码与模块对应

| 文件 | 对应模块 |
|---|---|
| `src/grammar.h` | 文法定义 + 数据结构 |
| `src/scanner.c` | 词法分析 |
| `src/parser.c` | 语法分析 + 语义分析 |
| `src/symbol.c` | 符号表 |
| `src/quadruple.c` | 中间代码 |
| `src/optimize.c` | 优化 |
| `src/codegen.c` | 目标代码生成 |
| `src/main.c` | 其它 (GUI/CLI) |

## 运行

```
make          # 编译
make test     # 测试 3 个样例
./compiler test/sample1.txt          # CLI 编译 → 生成 .s 汇编
gcc -no-pie test/sample1.txt.s -o s && ./s   # 汇编+运行
```
